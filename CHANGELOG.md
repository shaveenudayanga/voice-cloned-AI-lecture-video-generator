# Changelog

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-06-30

### Fixed

- **Fresh clone would fail to build** — the frontend Dockerfile previously assumed `node_modules/` was pre-installed on the host and copied it in (`COPY node_modules ./node_modules`). Since `node_modules/` is gitignored, a fresh `git clone` followed by `make first-run` would fail at the Docker build step with `COPY failed: file not found`. The Dockerfile now runs `pnpm install --frozen-lockfile` inside the builder stage, which is the correct approach. A `frontend/.dockerignore` was added to prevent the host `node_modules/` from inflating the build context on developer machines.

### Added

- **Upload PDF or PowerPoint slides** — drag-and-drop upload with automatic conversion; supports files up to 50 MB; delete and re-upload before continuing.
- **Record your voice once, reuse across all future lectures** — a single 60-second recording is saved as a voice profile and automatically selected for every new project; no re-recording required.
- **AI-generated narration scripts in your own speaking style** — each slide's script is generated by a multimodal AI that sees the actual slide image and mirrors your vocabulary, phrasing, and grammar by learning from your voice recording transcript.
- **Review and edit every slide's script before synthesis** — two-pane per-slide editor with explicit save per slide; regenerate a single slide's script without touching the rest.
- **Voice preview after recording** — a short test sentence is synthesized immediately after recording so you can confirm the clone quality before generating a full lecture.
- **Per-slide audio review** — play back each synthesized audio clip individually; re-synthesize only the slides whose scripts changed (unchanged slides are skipped automatically).
- **Automatic narrated video assembly with subtitles** — ffmpeg assembles all slides and audio into a single MP4; each slide is displayed for exactly the duration of its narration audio; an SRT subtitle file is included.
- **Download the finished video** — the completed MP4 is available to download directly from the browser.
- **Fully self-hosted** — all processing (voice cloning, AI script generation with a local model, video assembly) runs on your own hardware; no data leaves your machine unless you opt into the Gemini API for script generation.
- **Full observability for self-hosters** — structured JSON logs with correlation IDs, Prometheus metrics (queue depth, task duration histograms, cache hit rates), and OpenTelemetry distributed tracing from browser request to background worker.
